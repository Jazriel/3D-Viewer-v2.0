class MyApp.forms.UploadForm(formdata=<object object>, **kwargs)¶

Bases: flask_wtf.form.FlaskForm

It creates a form made for uploads.

It creates a form with a file field selector that requires a file to be selected in order to submit, and that only accepts files with an extension that is defined as “ALLOWED_MODEL_EXTENSIONS” into the config file. It also has a submit button.
