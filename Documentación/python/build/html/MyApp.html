MyApp.views.preview(filename)¶

Try to get the thumbnail of a model.

It can find the thumbnail if it has the same name of the model and the ‘.png’ extension.

filename : str
The filename of the model we want the thumbnail of.
flask.Response
Whenever the file exists:
-If there is a custom thumbnail for it, return it. -If there isn’t, return a default one.

If the file doen’t exist, it response with a 404 error.
